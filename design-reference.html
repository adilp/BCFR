Accessibility Features

Large Touch Targets

All interactive elements have minimum 44x44px touch targets for easy interaction on mobile devices and for users with motor impairments.

High Contrast

Text colors meet WCAG AA standards for contrast ratios, ensuring readability for users with visual impairments.

Focus Indicators

Clear focus indicators help keyboard users navigate the interface. Try tabbing through elements to see the blue outline.

Readable Typography

Larger base font size (17px) and generous line height make content easy to read for users of all ages.
